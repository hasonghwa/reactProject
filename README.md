<h1>🧨개인프로젝트: 옷 선택 시간 줄이기</h1>
<br>
<h2>👚자신이 입었던 옷을 등록해 시간을 절약하자!</h2>

평소에 어떤 색의 옷을 입을가에 대하여 시간이 오래 걸리시는 분들! 어떻게 입어야 상/하의 매치가 자연스러운지 헷갈리시는 분들! 이 앱을 사용해보세요! 당신의 시간을 절약해 줄 것 입니다!
<hr>
<h3>🎈이 앱은 다음과 같은 장점이 있습니다!</h3>
<br>
<ul> 
  <li>부위(ex 상의, 하의)를 정해 옷의 색깔을 정해 줄 수 있습니다.</li>
  <li>내가 등록했던 게시물을 참고하여 마음에 들었던 스타일을 다시 입을 수 있습니다.</li>
  <li>게시물의 선택/삭제가 가능합니다.</li>
  <li>게시물 등록에서 사진등록이 자유롭습니다.</li>
  <li>친구가 입었던 스타일을 참고하여 입는 것이 가능합니다.</li>
  <li>내 소개를 자유롭게 수정 할 수 있습니다.</li>
</ul>

<h3>👀간단한 앱 소개</h3>
<br>
  1. 로그인/회원가입
  <ul>
    <br>
    <li>사용자의 정보를 입력하고 로그인 합니다.</li>
  </ul>
<br>
  
   2-(1) 친구 피드
  <ul>
  <li>친구가 올린 게시물을 볼 수 있습니다.</li>
  </ul> 
<br>

  2-(2) 마이페이지
  <ul>
  <li>자신의 정보를 볼 수 있습니다.</li>

  </ul>
  <br>
  
  2-(3) 옷장 등록
  <ul>
  <li>원하는 선택지를 골라 게시글 등록이 가능합니다.</li>
  </ul>

<br>
  
  2-(4) 내 옷장
  <ul>
  <li>내가 올린 게시물들을 볼 수 있습니다.</li>

  </ul>

<h3>👢후기</h3>
두번째로 한 개인프로젝트 입니다. 개인적으로 저는 html보다 react가 코드의 자유성이 있어서 편하게 작업할 수 있었습니다.
자율성이 큰 만큼 코드는 몇배는 더 배우기 어렵다는 소문이 있었지만 저는 다른 문법보다 더 배우기 쉽다고 느껴졌습니다.
무엇보다 프론트와 백엔드 코드가 한 폴더에 다 같이 있었기에 페이지 이동을 여러 번 하지 않아도 되서 편리했습니다.
css, DB설계 등 다른 능력도 향상 되었지만 무엇보다 전과 확실히 비교되는 코딩실력이 보여서 뿌듯했습니다.
그 밖에 계획은 AI로 옷 핏을 확인 해주는 기능을 추가할 수 있으면 추가할 계획이 있습니다.
